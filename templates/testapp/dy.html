Data Type :

-------------------


Data Type represent the type of data present inside a variable. In Python we are not required to specify the type explicitly. Based on value provided,the type will be assigned automatically.Hence Python is Dynamically Typed Language.

Python contains the following inbuilt data types :

1. int
2. float
3.complex
4. bool
5. str
6. list
7. tuple
8. set
9. dict

Python contains several inbuilt functions :


1. type() :

to check the type of variable


2. id() :

to get address of object


3. print() :

to print the value


In Python everything is object

int data type :

We can use int data type to represent whole numbers (integral values)


code

>>> a=10

>>> print(a)

>>> print(type(a))

>>> print(id(a))


# 10

# int

# some address of int object


Complex Data Type :

A complex number is of the form


ex :

a+bj

here a is real part and bj is imaginary part

code

>>> a=2+3j

>>> print(a)

>>> print(type(a))

>>> print(id(a))


# 2+3j

# complex

# some address of complex object

We can use complex type generally in scientific Applications and electrical engineering Applications.


bool data type :

We can use this data type to represent boolean values.


The only allowed values for this data type are : True and False

Internally Python represents True as 1 and False as 0

code

>>> a=True

>>>a1=True+True

>>> a2=False+False

>>> print(a1)

>>> print(a2)

>>> print(a)

>>> print(type(a))

>>> print(id(a))


# 2

# 0

# 10

# bool

# some address of bool object

str type :

str represents String data type

A String is a sequence of characters enclosed within single quotes or double quotes


code

>>> a="muna"

>>> print(a)

>>> print(type(a))

>>> print(id(a))


# muna

# str

# some address of str object


float data type :

We can use float data type to represent floating point values (decimal values)


code

>>> a=12.42

>>> print(a)

>>> print(type(a))

>>> print(id(a))


# 12.42

# some address of float object
