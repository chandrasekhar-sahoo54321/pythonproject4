Polymophism :-

=========================



Poly means many. Morphs means forms.

Polymorphism means 'Many Forms'.

Yourself is best example of polymorphism.In front of Your parents You will have one type of behaviour and with friends another type of behaviour.Same person but different behaviours at different places,which is nothing but polymorphism.

The Same method with different implementations in Parent class and child classes.(overriding)

Polymorphism is 2 types :-

1. Method Overloading

2. Method Overriding

Method Overloading :-

================================


The concept of defining multiple methods with the same name or same number of parameter , different no of parameters within the same class is known as Method Overloading.

Example :-

=========================


class A:

def f1(self):

print("hello from f1")

def f2(self,a):

print("hello from f2")

def f3(self,a,b):

print("hello from f3")

a1=A()

a1.f1()

a1.f2(10,20)


If two method(whether both are inherited or both are declared , one is inherited and another one is declared)where same name but signature are not same is known as method overloading.

Method Overriding

================================

If two method(whether both are inherited or both are declared, one is inherited and another one is inherited) have the same name same signature but different implemetation is known as Method Overriding.

Example :-

==========================

class A:

def f1(self):

print("hello from f1 of A")

class B(A):

def f1(self):

print("hello from f1 of B")

b1=B()

b1.f1()
